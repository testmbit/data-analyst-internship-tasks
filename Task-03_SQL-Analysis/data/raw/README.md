# Raw Data

This folder represents the raw input data used for Task-03.

- Source: Brazilian E-Commerce Public Dataset by Olist (Kaggle)
- Format: CSV files
- Files were downloaded locally and imported into SQLite
- Raw CSV files are intentionally not committed to this repository

Reason:
Raw datasets are excluded to avoid duplication, large file sizes,
and licensing concerns. Only the final analytical SQLite database
is version-controlled.
