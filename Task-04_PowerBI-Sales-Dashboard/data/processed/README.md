# Processed Data

## Overview
This directory contains the cleaned and pre-processed sales dataset
used directly for building the Power BI dashboard.

The dataset is structured and ready for analysis and visualization.

## Dataset File
- **File name:** `cleaned_sales_data.csv`

This file contains sales-related fields such as:
- Sales amount
- Order identifiers
- Product information
- Time-based attributes (month, quarter, year)

The dataset was suitable for KPI calculation and trend analysis without
requiring additional cleaning within this project.

## Usage
The processed dataset was used to:
- Create KPI summary cards (Total Sales, Orders, Average Order Value)
- Perform time-series analysis (monthly and quarterly trends)
- Support interactive filtering within the Power BI dashboard

## Data State
- Status: **Cleaned / Processed**
- Transformation location: External (prior to this project)

## Source
Sales / Financial Dataset (Kaggle)
