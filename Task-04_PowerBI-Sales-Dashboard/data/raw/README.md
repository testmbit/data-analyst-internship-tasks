# Raw Data

## Overview
This directory is intended to store the original, unmodified raw dataset
used for analysis and dashboard creation.

## Data Availability
The raw dataset is **not available** for this task.

The Power BI dashboard was built using a **cleaned and pre-processed sales dataset**
that was already prepared prior to dashboard development.

## Notes
- No transformations were applied within this repository on raw data.
- All analysis and visualization were performed using the processed dataset.
- This approach is common when working with pre-cleaned datasets provided
  for reporting or visualization purposes.

## Source
