# Dashboard Screenshots

## Overview
This directory contains static preview images of the interactive
Power BI sales dashboard created for Task 04.

## Available Screenshot
- **overview.jpg**

This image represents the complete dashboard layout, including:
- KPI summary cards
- Monthly sales trend visualization
- Quarterly sales breakdown

## Note on Interactivity
The Power BI dashboard includes interactive features such as slicers
and filters that cannot be fully represented in static images.

For full interactivity and detailed exploration, refer to the
Power BI `.pbix` file located in the `dashboard/` directory.
