# Reports

This directory contains presentation material summarizing the
interactive Power BI sales dashboard created for Task 04.

The report is designed for business stakeholders and provides
a concise, executive-level summary of dashboard findings.

## Available Report

The presentation includes:
- Dashboard snapshot
- Key performance indicators (KPIs)
- Data-driven insights
- Actionable business recommendations

presentations, reviews, and stakeholder discussions.
